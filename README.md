# Lexicon-Based Sentiment Analysis 

A simple Natural Language Processing (NLP) project that performs sentiment analysis using a **lexicon-based approach** with NLTK.

This project classifies text into **Positive**, **Negative**, or **Neutral** based on the Opinion Lexicon.

---

##  Features

* Text preprocessing and tokenization
* Lexicon-based sentiment scoring
* Positive / Negative / Neutral classification
* Lightweight and easy to understand
* Suitable for NLP beginners

##  How It Works

1. The text is cleaned and tokenized
2. Words are matched against the Opinion Lexicon
3. A sentiment score is computed
4. The final sentiment is predicted
